I enjoy creating things

But what might creating things entail? My process looks a little something like this for 3D products:

  1. Write the idea down on paper.
  2. Create a sketch of what it might look like.
  3. Measure everything needed for the particular idea.
  4. Make a CAD version using Fusion360.
  5. 3D print the CAD model.
  6. Test in real life.

What about if it were a video idea? For that it looks a little more like this:

  1. Write the idea down on paper. (Yes the same.)
  2. Create a storyboard outlining the main sections.
  3. All assets needed for the video created in Krita, Inkscape or Blender
  4. Edit video clips together according to storyboard.
  5. Add in the previously created assets.
  6. Adjust audio levels to avoid clipping.
  7. Export the finalised video.
